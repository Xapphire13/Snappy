# Snappy
Filling the gaps in Windows Snap Assist

## Features
* Snap to top half of screen - <kbd>WIN</kbd> + <kbd>ALT</kbd> + <kbd>&uarr;</kbd>
* Snap to bottom half of screen - <kbd>WIN</kbd> + <kbd>ALT</kbd> + <kbd>&darr;</kbd>
